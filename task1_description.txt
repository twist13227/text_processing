Проверено каждое из ограничений из условия;
Объединены в один regexp с помощью использования positive lookahead checks

2. Проверка корректности web цвета
Для каждой из 3х кодировок цвета написан regexp "в лоб".
При помощи '|'.join они объединены в один

3. Токенизация математического выражения
Написано "в лоб" по условиям
(для каждого варианта элемента написан свой regexp, при помощи '|'.join они объединены в один)

Столкнулся с тем, что сначала должна быть проверка на константы и функции,
потому что иначе они считаются за переменные, что заведомо неверно
